Mesh Player is a FREE (yes you heard me) high-performance local audio client designed around the native Apple Music aesthetic but supercharged for local file management, spatial audio parsing, and advanced metadata visualization.

Core Architecture & Features
Zero-Config Media Import: Automatically crawls standard local file trajectories (Home > Music > Music > Media > Music) to surface your existing offline catalog instantly.

Native Apple Music Integration: Directly reads and imports your existing Apple Music app playlists while keeping your local library organized.

Spatial Audio Awareness: Built-in metadata scanning to natively identify and display Dolby Atmos asset tags.

High-Fidelity Lyric Engine: Custom parsing engine optimized for synchronized lyric formats. It maps the timeline via explicit [XX.XXX] time markers to handle smooth vertical scrolling, line highlighting, and automatic instrumental breaks.

Enhanced Layout & Flexibility:

Advanced Sorting: View your songs sorted chronologically by Date Added.

Versatile UI States: Switch fluidly between the default catalog explorer, a compact background miniplayer, and an immersive Fullscreen Player mode.

Adaptive Theme Engine: Features fluid UI themes, including a dynamic, hardware-accelerated ambient mesh gradient that matches the color profile of the active track's artwork.
